sChart.js 中文文档

sChart.js：一个小型简单的图表库。该文档介绍了该库的所有知识。
 立即下载

开始使用

引入js文件

首先我们需要在页面中引入Chart.js文件。

HTML
  1. <script src="sChart.js"></script>
创建画布

为了创建图表，先要生成一个画布，通过画布的id去实例化一个图标对象。

HTML
  1. <canvas id="canvas" width="500" height="400"></canvas>
创建图表

通过实例化sChart对象来生成图表。

JavaScript
  1. new sChart(canvasId, type, data, options)

完成实例化之后，就可以在画布上看到生成的图表了。

柱状图（Bar chart）

使用案例
数据结构 
JavaScript
  1. var data = [
  2.     {name:'2012', value:1141},
  3.     {name:'2013', value:1499},
  4.     {name:'2014', value:2260},
  5.     {name:'2015', value:1170},
  6.     {name:'2016', value:970},
  7.     {name:'2017', value:1450}
  8. ]
创建图表 
JavaScript
  1. new sChart('canvas', 'bar', data, {
  2.     title: '某商店近年营业总额',
  3.     bgColor: '#829dda',
  4.     titleColor: '#ffffff',
  5.     fillColor: '#72f6ff',
  6.     axisColor: '#eeeeee',
  7.     contentColor: '#bbbbbb'
  8. });
图表参数 
JavaScript
  1. options = {
  2.     padding: 50,                    // canvas 内边距
  3.     yEqual: 5,                      // y轴分成5等分
  4.     bgColor: '#ffffff',             // 默认背景颜色
  5.     fillColor: '#1E9FFF',           // 默认填充颜色
  6.     axisColor: '#666666',           // 坐标轴颜色
  7.     contentColor: '#eeeeee',        // 内容横线颜色
  8.     title: '',                      // 图表标题
  9.     titleColor: '#000000',          // 图表标题颜色
  10.     titlePosition: 'top'            // 图表标题位置: top / bottom
  11. }

饼状图（Pie chart）

使用案例
数据结构 
JavaScript
  1. var data = [
  2.     {name:'短袖', value:1200},
  3.     {name:'休闲裤', value:1222},
  4.     {name:'连衣裙', value:1283},
  5.     {name:'外套', value:1314},
  6.     {name:'羽绒服', value:2314}
  7. ]
创建图表 
JavaScript
  1. new sChart('canvas', 'pie', data, {
  2.     title: '某商店各商品年度销量',
  3.     bgColor: '#829dca',
  4.     titleColor: '#ffffff',
  5.     legendColor: '#ffffff'
  6. });
图表参数 
JavaScript
  1. options = {
  2.     bgColor: '#ffffff',             // 默认背景颜色
  3.     title: '',                      // 图表标题
  4.     titleColor: '#000000',          // 图表标题颜色
  5.     titlePosition: 'top'            // 图表标题位置: top / bottom
  6.     legendColor: '#000000',         // 图例文本颜色
  7.     legendTop: 40,                  // 图例距离顶部的长度
  8.     colorList: ['#1E9FFF', '#13CE66', '#F7BA2A', '#FF4949', '#72f6ff'],    // 饼图颜色列表
  9.     radius: 100,                    // 饼图半径
  10. }
